# Web-Dev-PHP-Project
PHP project for software development class.

# Rquirements
  * Docker
  * Docker Compose

# How does it work?
1. Clone the repo:
    * HTTP:
       ```sh
       git clone https://github.com/gera9/Web-Dev-PHP-Project.git
       ```
    * SSH
       ```sh
       git clone git@github.com:gera9/Web-Dev-PHP-Project.git
       ```
2. Run Docker Compose:
   ```sh
   docker compose up
   ```
